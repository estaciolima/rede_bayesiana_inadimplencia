# Rede bayesiana inadimplência

Projeto final da disciplina de Redes Bayesianas do curso de pós-graduação em Estatística e Ciências de Dados da Universidade Federal da Bahia, ministrado pela

> Professora: Lilia Costa.
>
> Alunos: Estácio Lima e Gerson Júnior.

## Estrutura dos codigos

1.  `code/AMOSTRA.R`: Código para amostragem dos dados originais e configuração do ambiente
2.  `code/EDA.R`: Código para análise exploratória dos dados, definição da target e seleção.
3.  `code/comparacao_LR_BN.R`: Código para comparação entre modelos de regressão logística e redes bayesianas.
4.  `code/funcoes.R`: Funções auxiliares utilizadas nos outros códigos.
